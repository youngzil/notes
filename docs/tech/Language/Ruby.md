1、Ruby on Rails
Ruby on Rails是一个可以使你开发、部署、维护 web 应用程序变得简单的框架。这个 Web 开发框架在开发人员当中极其流行，它声称“为确保编程员满意和持续高效地工作进行了优化”。用户包括 Basecamp、推特、Shopify 和 GitHub 等公司。

支持的操作系统：Windows、Linux 和 OS X
相关网站：https://rubyonrails.org
